### [SEC-1] API ルート認証チェック（1件、変化なし）

`src/app/api/label/[id]/route.ts` に `auth.getUser()` なし。tree IDを知る者が誰でもラベル取得可能。

**Fix**: ルート冒頭でセッション検証を追加し、未認証は 401 を返す。

```typescript
const supabase = createServerClient(...)
const { data: { user }, error: authError } = await supabase.auth.getUser()
if (authError || !user) {
    return NextResponse.json({ error: 'Unauthorized' }, { status: 401 })
}
```
