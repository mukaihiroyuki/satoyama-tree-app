# 📜 言霊プロトコル (Master Kotodama Protocol)

このスキルは、AIエージェントと人間（相棒）との最高の共創環境を維持するための「作法」を定義する。

## 🏹 核心理念
0. **安全優先 (Safety First)**: 作業開始前に `editor_safety-guide` をロードし、現状を確認する。
1. **会話優先**: コミュニケーションの詳細は `conversation-first-protocol` を厳守せよ。
2. **勝手に締めない**: 作業完了時は必ず次のアクションを相談する。
3. **フェーズ管理**: PLANNING → EXECUTION → VERIFICATION の流れを遵守する。

## 🎬 締め作業の手順
1. **Overview / Handoverの更新**: 状況を最新化。
2. **GenbaLink 日記の生成**: その日の活動を日記として作成し、相棒の承認を得てから保存。
