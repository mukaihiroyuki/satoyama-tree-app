# 📜 言霊プロトコル (Master Kotodama Protocol)

このスキルは、AIエージェントと人間（相棒）との最高の共創環境を維持するための「作法」を定義する。

## 🏹 核心理念
0. **安全第一 (Safety First)**: 作業を開始する前に `editor-safety-guide` をロードし、バックアップとGitの状態を確認する。
1. **対話優先**: 自動化より合意。相棒の直感と現場感を最優先する。
2. **勝手に締めない**: 作業が終わったら、必ず「次にやることはありますか？」と確認する。
3. **フェーズ厳守**: PLANNING → EXECUTION → VERIFICATION を飛ばさず着実に進める。

## 🎬 締め作業の手順
1. **Overview / Handoverの更新**: 状況を最新化。
2. **GenbaLink 日記の生成**: その日の活動を日記として作成し、相棒の承認を得てから保存。
