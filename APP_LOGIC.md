# APP_LOGIC: 里山樹木管理アプリ

## 🌳 概要
圃場（ストックヤード）での樹木在庫を管理するためのNext.js PWA。モバイルファーストの入力、QRラベル発行、Supabaseとのリアルタイム同期に焦点を当てる。

## 🛠 技術スタック
- **フレームワーク**: Next.js 15 (App Router)
- **データベース/認証**: Supabase
- **スタイリング**: Tailwind CSS 4
- **印刷**: AirPrint (Brother RJ-4250WBを想定)

## 🚀 実装済み機能
- [x] **ダッシュボード**: 基本統計（総数、在庫数、総額）。
- [x] **樹木登録**: 新規樹種の登録フォーム（樹種、樹高、価格、メモ）。
- [x] **樹木一覧**: 全データ表示とフィルタリング（状態、樹種、場所）。
- [x] **樹木詳細**: 単体データの詳細表示 ＋ 写真アップロード。
- [x] **写真アップロード**: 画像の圧縮（1200px）とSupabase Storage連携。

## 🔜 未実装 / 今後の予定
- [ ] **QRラベル生成**: 樹木ID/URLを含む印刷用ラベルの生成。
- [ ] **QRスキャナー**: カメラを使用したラベル読み取りと詳細画面への遷移。
- [ ] **オフライン対応**: PWA化（Service Workers）とオフラインデータ入力。
- [ ] **Googleスプレッドシート同期**: 管理用スプレッドシートへの自動同期。
